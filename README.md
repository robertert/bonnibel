# Bonnibel

System zarządzania postępem prac zespołu — integruje Jira, Git i dokumentację
w jednym miejscu. Projekt zespołowy inżynierii oprogramowania, sem. letni 2025/2026.

## Struktura repozytorium

```
.
├── bonnibel-frontend/   # React + TypeScript + Vite + Tailwind (szkielet)
└── bonnibel-backend/    # placeholder — stack do zdecydowania
```

Każdy podprojekt ma własne `README.md` z instrukcją i opisem stanu.
